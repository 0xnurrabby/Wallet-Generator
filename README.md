<i>A browser-based EVM wallet batch generator .... create 1 to 1000 wallets locally with addresses, private keys, and seed phrases, then export as TXT, JSON, or CSV.</i>
</div>

---

## ✦ Features

<div align="center">

| | Feature | What it does |
|:---:|---|---|
| 🔑 | Batch generation | Generate 1 to 1000 EVM wallets in a single click |
| 📋 | Bulk copy sections | Separate bulk copy for all addresses, all keys, all phrases |
| 📄 | Export formats | Export the full batch as TXT, JSON, or CSV |
| 🔤 | Phrase length options | Choose 12 / 15 / 18 / 21 / 24 word seed phrases |
| 🛤️ | Custom derivation path | Change the derivation path base before generating |
| 📎 | Per-wallet copy | Copy individual address, key, or phrase with one click |
| 🔒 | Local only | Everything runs in the browser, nothing is sent to a server |

</div>

## ✦ Download & Run

**Step 1** .... Clone the repo

```bash
git clone https://github.com/0xnurrabby/Wallet-Generator
cd Wallet-Generator
```

**Step 2** .... Open in browser

```bash
start index.html
# No build or server needed
```

**Step 3** .... Generate wallets

---

## ✦ Setup

```
1. Clone the repo or download as ZIP
2. Open index.html in any modern browser
3. Enter the number of wallets to generate (1 to 1000)
4. Choose phrase length (12 / 15 / 18 / 21 / 24 words)
5. Optionally change the derivation path base
6. Click "Generate wallets"
7. Use bulk copy buttons or per-wallet copy buttons as needed
8. Export as TXT, JSON, or CSV if you need the full list

IMPORTANT: This tool displays private keys and seed phrases in plain text.
Only use it on a trusted device and handle any exports carefully.
```
